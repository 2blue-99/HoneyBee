# HoneyBee 허니비

## 개요
- 서버, 웹, DB, 앱 개발자들과 합작한 연구실 페이지 제작 프로젝트 - 결제 영수증 관리 어플리케이션



## 개발 동기
- 과제 카드로 연구실 인원들의 식사 시, 결제 영수증을 따로 관리해야 하는 불편함 발생.
연구실 페이지 제작 프로젝트를 기회삼아 웹, 서버, 앱을 통해 효과적으로 관리되는 영수증 관리 서비스 앱 개발.


## 기간
- 2023.02 ~ 20.23.04


## 기술 스택
- Kotlin
- MVVM
- Clean Architecture
- Dagger Hilt
- Jetpack Navigation component
- Room Database
- Lottie
- Coroutine
- Retrofit
